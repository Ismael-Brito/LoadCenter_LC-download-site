# 📦 Load Center

Automação para download automatizado de arquivos do portal Brasíndice, com interface gráfica e controle de progresso em tempo real.

---

## 🚀 Funcionalidades

- ✅ Login automático (com sessão via cookies)
- ✅ Download automático por edição
- ✅ Organização automática em pastas por edição
- ✅ Barra de progresso na interface
- ✅ Interface gráfica simples e intuitiva

---

## ▶️ Como Executar

Baixe o instalador no link [LoadCenter](https://ismael-brito.github.io/LoadCenter_LC-download-site/)


1. Informe a edição desejada
2. Clique em **Iniciar**
3. Aguarde os downloads concluírem

Os arquivos serão salvos em:

  Área de trabalho/loadcenter/NÚMERO DA EDIÇÃO/


---

## 🔐 Sobre Login

- A sessão é salva temporariamente via cookies.
- Caso a sessão expire, o login é feito automaticamente.
- O arquivo de cookies é removido após uso.

---

## 📈 Barra de Progresso

A interface mostra:

- Processo atual sendo executado
- Barra de progresso
- Status final

---

## 📌 Requisitos

- Windows 10+
- Microsoft Edge instalado
- Edge WebDriver compatível com sua versão
